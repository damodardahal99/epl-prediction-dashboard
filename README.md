# ⚽ EPL 2025–2026 Prediction Accuracy Dashboard

A Streamlit web app that compares **user‑submitted Premier League predicted standings** against the **actual EPL standings** scraped live from ESPN.

The app calculates:
- Per‑team prediction errors  
- Total error per predictor  
- Predictor ranking (best → worst)  
- A visual accuracy chart  
- Team logos for a clean UI
